Augmented Reality Experience





Description

This was a college assignment I created for my Virtual and Augmented reality module. This assignment tested my ability to create an interactive Augmented Reality based experience using Unity3D and Vuforia. It is a short unity based experience which features a zombie that can be controlled to walk and attack. To use this you need to download the APK which is provided.



Video



Instructions

First you need to download the APK. You also need to enable downloads from unknown devices to run the APK. Then run the APK and view the poster (AR Marker) with your phone. You can use the virual joystick to move the zombie around and press the attack button to make him attack.
